- [x] Create Account
- [x] Request Secret

- [x] Like / Unlike a photo

- [ ] See user profile
- [ ] Follow / Unfollow user
- [ ] See the full photo
- [ ] Edit my profile
- [ ] Upload a photo
- [ ] Edit the photo
